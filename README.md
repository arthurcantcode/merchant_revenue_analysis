# Merchant Revenue & Data Integrity Analysis

This project demonstrates a comprehensive data audit and revenue analysis of a merchant dataset. 
The goal was to transform raw, inconsistent data into actionable financial insights.

### Project Objectives
  1. **Data Integrity Audit:** Identified and resolved over 600 issues including duplicates, label inconsistencies, and missing revenue entries.
  2. **Segment Growth Analysis:** Analyzed revenue trends across "Growth Engine" tiers (Emerging, Corporate, Enterprise).
  3. **Regional Performance:** Evaluated market share across Business Units (UK, ANZ, France, DACH, North America).

### 🛠 Tools & Techniques
* **Excel (IF, COUNTIFS, COUNTA, OR and Logic):** Used to map segments and perform multi-column cross-checks for data quality.
* **Pivot Tables and Graphs:** Built to investigate and present different insights extracted from the clean data.
* **Data Cleaning:** Implemented a "Clean Data Switch" system to filter out corrupted entries for final reporting.

### 📂 Key Files
* **[Merchant_Revenue_Analysis.xlsx](./Merchant_Revenue_Analysis.xlsx):** The full interactive model including data quality summaries and pivot analysis.
* **[Raw_Data_Preview.csv](./Merchant_Revenue_Analysis.xlsx%20-%20Raw_Data_Analysis.csv):** A snapshot of the audit process showing the error-checking logic.

### 📈 Key Insights from Dummy Data
* **Market Concentration:** The UK represents 78% of all contracts, acting as the primary revenue engine.
* **Stable Growth:** Identified an average annual revenue growth of ~7.6% from FY23 to FY25.
* **Segment Breakdown:** "Emerging" merchants account for 63% of total revenue, highlighting a strong small-business focus.
